== Description ==
LitheStore is a free elegant E-Commerce Theme that you can use it to build your online shop easily and quickly.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Lithestore in the search form and press the 'Enter' key in your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel.
